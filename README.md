 Plantec – Sistema de Automação IoT para Cultivo

 Descrição
Projeto de automação IoT desenvolvido para controle de irrigação, adubação e acionamento de sombrite, utilizando ESP32 e comunicação via protocolo MQTT.

O sistema opera em modo manual e automático, permitindo monitoramento e controle remoto.

 Tecnologias Utilizadas
- ESP32
- Protocolo MQTT
- IoT
- Sensores e atuadores
- Comunicação serial UART

Funcionalidades
- Controle automático de irrigação
- Controle de adubação
- Acionamento de sombrite
- Monitoramento remoto
- Operação manual e automática

 Status do Projeto
Projeto acadêmico concluido.
